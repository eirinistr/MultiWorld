## Repository Structure

The Python files contain the code and prompts used throughout all experiments presented in the thesis.

* **Monolingual experiments:** Each Python file contains the implementations of all three prompting techniques (Zero-Shot, Few-Shot, and Chain-of-Thought).
* **Multilingual experiments:** Each Python file contains the prompting strategy used in the corresponding multilingual experiment.
* **Monolingual answer categorization** and **Multilingual answer categorization:** These files contain the prompts provided to Gemini for the automatic categorization of model responses.
* **Plotting:** Contains the code used to generate all figures and visualizations. Some of the generated plots are included in the thesis, while others were created during the analysis phase and are not presented in the final document.

The repository accompanies the thesis and provides the material required to reproduce the experimental pipeline and analysis.
